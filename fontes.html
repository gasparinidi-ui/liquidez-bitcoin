Sobre o fluxo de ETFs (atualização manual)

A Farside Investors protege a página com Cloudflare, que bloqueia requisições automatizadas mesmo com cabeçalhos de navegador — isso é comum em sites que querem evitar scraping em massa. Para manter o pipeline 100% gratuito, esse indicador fica marcado como atualização manual: confira o valor mais recente direto em farside.co.uk/btc e, se quiser, atualize data_live.json manualmente com a chave btc_etf_netflow.

Arquitetura de atualização (custo zero)

Este site é 100% estático (HTML/CSS/JS) e não roda servidor. Os dados "ao vivo" são publicados assim:

EtapaO que faz
1. GitHub Actions (cron)Roda scripts/data_collector.py a cada hora (ou com a frequência que preferir), buscando os 30 indicadores nas fontes acima.
2. Commit automáticoO script grava data_live.json e o workflow faz commit direto no repositório.
3. Hospedagem estáticaVercel, Netlify, GitHub Pages ou Cloudflare Pages servem os arquivos — qualquer um dos planos gratuitos serve.
4. Leitura no navegadorassets/data_loader.js busca data_live.json a cada carregamento de página e preenche os valores.

Frequência máxima gratuita: em repositórios públicos, o GitHub Actions tem minutos ilimitados — o cron pode rodar a cada 5–15 minutos sem custo. O limite prático é a própria frequência de publicação de cada fonte: a maioria dos dados macro (Fed, BCE, M2) só muda 1x por dia ou por semana; os dados de liquidez cripto (funding rate, stablecoins, ETF flow) fazem sentido em intervalos de 15–60 minutos.

Como publicar (passo a passo)

  1. Crie um repositório público no GitHub e suba esta pasta inteira.
  2. Obtenha uma chave gratuita do FRED em fredaccount.stlouisfed.org/apikeys (aprovação instantânea).
  3. No repositório: Settings → Secrets and variables → Actions → adicione o secret FRED_API_KEY.
  4. O workflow .github/workflows/update-data.yml já está configurado para rodar de hora em hora e commitar data_live.json.
  5. Conecte o repositório à Vercel (ou Netlify/Cloudflare Pages) — deploy automático a cada commit, plano gratuito.
